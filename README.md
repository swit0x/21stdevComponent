# Next.js template

This is a Next.js template with shadcn/ui and 21st.dev component liberaies used 

## Adding components

To add components to your app, run the following command:

```bash
npx shadcn@latest add "Componentname" from 21st.dev 
```

This will place the ui components in the `components` directory.

## Using components

To use the components in your app, import them as follows:

```tsx
import { component } from "@/components/ui/button";

```
## Running project 

npm run build 

npm run dev
